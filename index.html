vIP.js 라이브러리 소개

소개

vIP.js는 JavaScript 기반의 라이브러리로, 웹 페이지에서 IP 주소 추적과 특별한 기능 제공을 간단하게 구현할 수 있습니다.

기능

  • IP 주소 추적
  • 특정 IP 주소에게 특별한 페이지 표시
  • 커스텀 특별한 페이지 내용 설정

사용법

아래의 예제 코드를 통해 vIP.js를 사용하는 방법을 확인하세요:

            
                // IP 주소 추적
                vIP.trackIP('192.168.1.1');
                vIP.trackIP('특정_IP_주소');

                // 특별한 페이지 내용 설정
                vIP.setSpecialPageContent('
커스텀 특별한 페이지
');

                // 특정 IP 주소에게 특별한 페이지 보여주기
                vIP.showSpecialPage('특정_IP_주소', 'specialPage');

라이선스

vIP.js는 Apache 2.0 라이선스로 배포되어 자유롭게 사용할 수 있습니다.
